الوحدات الحالية والمستقبلية

بنيت الصفحة بشكل يعكس رحلة من موقع تعريفي إلى نظام تشغيل داخلي مصغر. الوحدتان المطلوبتان تعملان الآن، مع إبراز مسار التوسع المستقبلي ليكون المشروع أساسًا فعليًا لمنصة الشركة.

01

وحدة الحضور والانصراف

الموظف يسجل الحضور أو الانصراف بضغطة واحدة، ويتم إنشاء سجل زمني تلقائي يمكن مراجعته وإدارته من لوحة الأدمن.

  • تسجيل دخول وخروج مباشر
  • حفظ السجلات محليًا كنسخة أولية
  • تعديل وحذف وتصدير من لوحة الأدمن
02

وحدة مسير الرواتب

الإدمن يضيف بيانات الراتب يدويًا لكل موظف ولكل شهر، ثم يمكن للموظف عرض آخر مسير خاص به وتنزيله مباشرة.

  • راتب أساسي وبدل سكن وبدل نقل
  • خصم التأمينات وصافي الراتب
  • اسم الموظف والشهر ضمن المسير
03

امتدادات ERP لاحقًا

تم إبقاء البنية مرنة لتوصيل خدمات إضافية مثل الإجازات، ملفات الموظفين، العقود، الموافقات، التقارير التنفيذية، والتنبيهات.

  • ربط لاحق بواجهة API وقاعدة بيانات
  • نظام صلاحيات أكثر تفصيلاً
  • تقارير ورسوم بيانية وعمليات أرشفة

لوحة الخدمات التفاعلية
